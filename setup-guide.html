🥛 VCG/MRG Identification App – Complete Setup Guide

✅ App is ready! Follow the steps below to set up Google Sheets sync and deploy the app on mobile devices.

📱 Part 1: Running the App on Mobile

Option A: Share as a file (simplest)

1 Copy the entire vcg-mrg-app folder to a shared location (Google Drive, WhatsApp, etc.)

2 On Android: Open Chrome → Navigate to the index.html file → Tap ⋮ menu → "Add to Home Screen"

3 On iPhone: Open Safari → Open the index.html → Tap Share → "Add to Home Screen"

Option B: Host on Google Drive (recommended for team)

1 Upload the vcg-mrg-app folder to Google Drive

2 Right-click index.html → Share → Anyone with link → Copy link

3 Share the link with all field executives via WhatsApp

4 Open in Chrome on phone → Add to Home Screen

Option C: Host on GitHub Pages (free, permanent URL)

1 Create a free GitHub account at github.com
2 New repository → Upload the vcg-mrg-app folder
3 Settings → Pages → Source: main branch → Save
4 App URL: https://yourusername.github.io/vcg-mrg-app/
5 Share this URL with all executives. Works on any phone!

☁️ Part 2: Google Sheets Sync Setup

This enables all field data to automatically sync to a central Google Sheet viewable by admin.

1 Go to sheets.google.com and create a new spreadsheet. Name it "VCG MRG Identifications 2025"
2 Copy the Spreadsheet ID from the URL:
https://docs.google.com/spreadsheets/d/YOUR_ID_HERE/edit
3 Go to script.google.com → New Project
4 Delete the default code and paste the entire contents of google-apps-script.js

Replace YOUR_GOOGLE_SHEET_ID_HERE with your actual Spreadsheet ID
5 Click Deploy → New Deployment
• Type: Web App
• Execute as: Me
• Who has access: Anyone
• Click Deploy → Authorize → Copy the Web App URL
6 In the app, go to ⚙ Setup → Paste the Web App URL → Save
✅ Done! Every identification submitted by field executives will now automatically appear in your Google Sheet.

Offline Capability

The app works 100% offline. All data is saved locally on the device. When internet is available, it automatically syncs to Google Sheets. The sync status is shown at the top of the dashboard.
